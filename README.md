该项目为javaFx开发的实用小工具集[xJavaFxTool](https://gitee.com/xwintop/xJavaFxTool)的插件。

访问地址：
```
<repositories>
	<repository>
		<id>xwintop-maven</id>
		<url>https://xwintop.gitee.io/maven/repository</url>
	</repository>
</repositories>
```

该库后续将废弃，请使用
```
<repositories>
	<repository>
    	<id>jitpack.io</id>
    	<url>https://jitpack.io</url>
	</repository>
</repositories>
```
